- ls : list files in directory
- pwd : print working directory
- mkdir: make new directory
- cd : change directory
- cd .. : up directory (parent directory)
- touch : create new file.
- cat : show file content without opening.
- mv my_directory new_location/ : to move directory.
- ctrl D to exist sudo mode
